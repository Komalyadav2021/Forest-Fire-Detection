# 🔥 Forest Fire Detection System

A deep learning-based system for detecting forest fires using image data. This project leverages Convolutional Neural Networks (CNNs) to classify satellite or drone images as either "fire" or "no fire".

---

## 📌 Tech Stack

- **Python**: Core programming language
- **TensorFlow / Keras**: For building and training the CNN model
- **OpenCV**: Image preprocessing (resizing, normalization)
- **NumPy / Pandas**: Data manipulation and preparation
- **Matplotlib**: Visualization of model performance
